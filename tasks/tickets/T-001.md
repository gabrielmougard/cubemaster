# T-001: Set up Embassy project skeleton and toolchain

**Epic:** EPIC-001 (MVP Firmware)
**Status:** Backlog
**Priority:** P0
**Estimate:** 3 days
**Labels:** `firmware`, `setup`, `embassy`

## Description

Create the Rust firmware crate inside the workspace (`firmware/`) with
the Embassy async executor, ESP32-S3 target configuration, and a
working "hello world" that blinks the onboard LED and prints to serial
via `defmt`.

## Tasks

1. Add `firmware/` member to workspace `Cargo.toml`
2. Configure `.cargo/config.toml` for `xtensa-esp32s3-none-elf`
3. Pin nightly toolchain in `rust-toolchain.toml`
4. Add Embassy deps: `esp-hal`, `embassy-executor`, `embassy-time`
5. Write `main.rs` with LED blink + `defmt::info!("Hello CubeMaster")`
6. Flash via `cargo run` and verify serial output
7. Document toolchain setup in README

## Acceptance Criteria

- `cargo build --release` produces a valid ELF binary
- `cargo run` flashes and runs on physical ESP32-S3 devkit
- Serial output shows "Hello CubeMaster" and LED blinks
- Binary size <200KB (baseline)
