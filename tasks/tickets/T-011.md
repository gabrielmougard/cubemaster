# T-011: Firmware integration test and end-to-end demo

**Epic:** EPIC-001
**Status:** Backlog
**Priority:** P0
**Estimate:** 5 days
**Labels:** `firmware`, `testing`, `integration`

## Description

Perform full integration testing of all firmware modules on the
breadboard prototype. Verify end-to-end scenarios: BLE upload → voice
detection → correct sound + LED playback. Measure performance metrics,
fix bugs, and prepare the demo.

## Tasks

1. Assemble full breadboard with all peripherals
2. Flash firmware and verify boot without errors
3. Test BLE advertisement and connection
4. Upload a sound file via BLE (small) and Wi-Fi (large)
5. Configure keyword "fire" → uploaded sound ID 0
6. Say "fire" and verify correct sound plays + LEDs animate
7. Test button actions (volume cycle, mute, deep sleep)
8. Measure firmware binary size (target <500KB)
9. Measure RAM usage at peak (target <60KB)
10. Run 2-hour soak test (continuous BLE + voice monitoring)
11. Document any bugs found
12. Record demo video showing end-to-end flow

## Acceptance Criteria

- Binary <512KB flash, <64KB RAM peak
- 2-hour soak test completes without crashes
- Voice detection accuracy ≥90% (quiet room)
- BLE upload completes without corruption
- LED synchronization lag <50ms from sound start
- Demo video recorded and archived
