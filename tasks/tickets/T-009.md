# T-009: Implement button handler

**Epic:** EPIC-001
**Status:** Backlog
**Priority:** P1
**Estimate:** 2 days
**Labels:** `firmware`, `buttons`, `gpio`

## Description

Implement the button handler module (`firmware/src/btn_mon.rs`) for
the power button (GPIO0) and volume button (GPIO1). Support single
press, long press (1–3s), and very long press (>3s) detection with
software debounce.

## Tasks

1. Configure GPIO0 and GPIO1 as input with internal pull-up
2. Set up GPIO interrupt on falling edge (press)
3. Implement software debounce (50ms window)
4. Implement press duration detection:
   - <1s: short press
   - 1–3s: long press
   - >3s: very long press
5. Create Embassy task: `btn_task(event_tx: Sender<BtnEvent>)`
6. Map button actions per spec (see spec table)

## Acceptance Criteria

- Short press (<1s) detected reliably
- Long press (1–3s) detected without false short triggers
- Very long press (>3s) detected
- No button bounce artifacts
- Binary size increase <2KB
