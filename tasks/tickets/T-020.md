# T-020: Implement LED pattern preview canvas

**Epic:** EPIC-002
**Status:** Backlog
**Priority:** P1
**Estimate:** 3 days
**Labels:** `companion-app`, `ui`, `led-pattern`

## Description

Implement an LED pattern preview canvas that renders a simulated
16×16 LED grid showing the current pattern as it would appear on a
cube face. The canvas should support procedural pattern previews and
imported bitmap patterns.

## Tasks

1. Create a 2D canvas component drawing a 16×16 pixel grid
2. Implement procedural pattern renderers (fire, plasma, rainbow)
3. Implement bitmap pattern renderer (from uploaded pattern data)
4. Animate patterns at ~15 FPS
5. Support zoom in/out (pixel-perfect at 100%, scaled up for detail)
6. Allow selecting which face to preview (0–3)
7. Add pattern switching dropdown
8. Integrate with the editor view for pattern assignment

## Acceptance Criteria

- Canvas renders a 16×16 grid with correct color mapping
- Fire pattern animation looks visually similar to cube behavior
- Preview updates when pattern is changed in dropdown
- Frame rate is smooth (≥15 FPS)
